Final Year Project Management System

• Designed and developed a Final Year Project Management System tailored for educational and professional environments, optimizing the management of industrial projects. Facilitated seamless coordination among distinct user roles, including Students submitting project titles and progress reports, Instructors evaluating submissions and providing feedback, a Reviewing Committee assessing project viability, and a Project Coordinator overseeing project status and advisor assignments. Enhanced efficiency, communication, and collaboration throughout project lifecycles, thereby streamlining Final year project management processes.
• Programming languages used was Java and SQL
